# MultiThreaded Web Server (Java)

This project demonstrates the implementation of a web server using three different approaches:
- Single-threaded server
- Multi-threaded server
- Thread pool-based server

It highlights how concurrent client requests are handled efficiently using Java multithreading.

---

## 🚀 Features
- Handles multiple client requests using sockets
- Comparison between single-threaded and multi-threaded models
- Thread pool implementation for optimized performance
- Demonstrates real-world server behavior

---

## 🛠️ Technologies Used
- Java
- Socket Programming
- Multithreading (Threads & ExecutorService)
